# Title
**AirBnB clone project**

## Description
Line-oriented command interpreters for AirBnB clone project

## Requirements
- Shell
- Python3

## Running the app
```bash
$ ./console.py
```

## Example
```bash
$ ./console.py
(hbnb)
(hbnb) help

Documented commands (type help <topic>):
========================================
EOF  help  quit
```
